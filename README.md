# Circle Back Prototype

A satirical, educational corporate-politics simulator with a full-screen terminal UI.

## Run

```bash
python3 -m pip install -r requirements.txt
python3 game.py
```

Use the inbox to choose a scene. Number keys select decisions, Enter continues
non-decision scenes, `i/c/a/h` switch dashboard tabs, and `q` quits.

The prototype campaign lasts 12 turns.
